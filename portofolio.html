RAG Haji dan Umrah

Saya mengembangkan aplikasi mobile RAG Haji & Umrah menggunakan Kotlin sebagai bagian dari tugas akhir. Aplikasi ini dirancang untuk membantu jamaah Haji dan Umrah dalam menjawab pertanyaan seputar ibadah dan prosedur perjalanan secara cerdas. Sistem menggunakan pendekatan Retrieval-Augmented Generation (RAG) dengan inferensi lokal berbasis LLaMA.cpp, sehingga mampu memberikan jawaban kontekstual tanpa memerlukan koneksi internet atau API eksternal. Fokus utama pengembangan adalah mengukur efektivitas penggunaan model secara lokal di perangkat terbatas seperti handphone Android.

Teknologi:

  • Kotlin – Pengembangan aplikasi mobile native Android
  • LLaMA.cpp – Inferensi AI lokal (tanpa API eksternal)
  • Retrieval-Augmented Generation (RAG) – Arsitektur pencarian dan generasi jawaban berbasis konteks
  • ObjectBox – Database lokal untuk penyimpanan data dan vector search
  • ONNX Runtime – Untuk menjalankan model embedding secara lokal di perangkat Android
  • Android Studio – Lingkungan pengembangan (IDE)

Link repository: https://github.com/ameliadewi19/RAGHaji.git

WABS (Whatsapp Broadcast Scheduler)

WABS, singkatan dari WhatsApp Broadcast Scheduler, adalah aplikasi inovatif untuk meningkatkan interaksi pengguna melalui WhatsApp. Dengan desain intuitif dan fungsionalitas canggih, WABS memungkinkan pengguna merencanakan dan mengirim pesan broadcast langsung atau terjadwal.

Teknologi:

  • Frontend: ReactJS
  • Backend: ExpressJS
  • Database: Postgre
  • Library: whatsapp-web.js, node-cron

Link repository: https://github.com/ameliadewi19/WABS
